# Known Faces Folder

Create one subfolder per employee using the employee name exactly as in backend records.

Example:
- `known_faces/Alice Johnson/face1.jpg`
- `known_faces/Alice Johnson/face2.jpg`
- `known_faces/Bob Smith/face1.jpg`

Notes:
- Use clear frontal face images.
- 3 to 10 images per person improves matching stability.
- Names are matched case-insensitively against backend employee names.
